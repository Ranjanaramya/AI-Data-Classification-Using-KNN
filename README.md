# Iris Flower Classification Using KNN

## Project Overview

This project implements a Machine Learning classification model using the Iris Flower Dataset. The model predicts the species of an iris flower based on sepal and petal measurements.

## Technologies Used

* Python
* Google Colab
* Scikit-Learn
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Machine Learning Workflow

1. Load Iris Dataset
2. Data Exploration
3. Train-Test Split
4. Feature Scaling
5. K-Nearest Neighbors (KNN) Classification
6. Model Evaluation
7. Confusion Matrix Visualization
8. Species Prediction

## Results

* Accuracy: 96%+
* F1 Score: 0.96+

## Dataset

The Iris Dataset contains 150 samples belonging to three species:

* Setosa
* Versicolor
* Virginica
